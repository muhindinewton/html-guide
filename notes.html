    • Chapter 1: Introduction to HTML

    üí° What is HTML?

    HTML stands for HyperText Markup Language.

    • HyperText means text with links to other documents
    • Markup means we "mark up" text to give it structure
    • Language means it has rules and syntax

    Important Things to Know:

    • ‚úÖ HTML is NOT a programming language‚Äîit's a markup language
    • ‚úÖ It tells your web browser what to display and how to organize content
    • ‚úÖ We are using HTML5, the latest version
    • ‚úÖ Every website you visit uses HTML!
    üåç Real-World Example:

    Think of HTML like the skeleton of a building:

    • The skeleton (HTML) gives structure
    • Paint and decorations (CSS) make it look beautiful
    • Electricity and plumbing (JavaScript) make it functional

    Chapter 2: Your First Webpage Structure

    üèóÔ∏è The Basic HTML Skeleton

    Every HTML page MUST have this structure. Memorize it!

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

    üìù Understanding Each Part:

    Tag Purpose
    <!DOCTYPE html> Tells the browser "This is HTML5"
    <html lang="en"> Root element. lang="en" means English language
    <head> Contains information ABOUT the page (not visible)
    <meta charset="utf-8"> Supports special characters like √°, √±, ‚Ç¨
    <title> Shows in browser tab
    <body> Everything users SEE goes here
    üí° Pro Tip: Always start EVERY HTML file with this skeleton!

    Chapter 2: CSS Basics
    

    
    
      
    What is CSS?
    
      
    CSS (Cascading Style Sheets) is used to style and layout web pages.
    
      
    Key concepts include:
    
      
      
        
    • Selectors - Target HTML elements to style
      • 
        
    • Properties - Define what to style (color, size, etc.)
      • 
        
    • Values - Define how to style (red, 16px, etc.)
      • 
      
    
    
    

    
    Basic CSS Syntax
    
    
    selector {
    property: value;
    another-property: value;
}
    

    
    Common CSS Properties
    
    
    
      
    /* Text Styling */
color: #333;
font-family: Arial, sans-serif;
font-size: 16px;
text-align: center;
text-decoration: none;

/* Box Model */
width: 100%;
height: 200px;
margin: 10px;
padding: 20px;
border: 1px solid #ddd;
border-radius: 5px;

/* Layout */
display: block;
position: relative;
top: 0;
left: 0;

/* Colors and Backgrounds */
background-color: #f8f9fa;
background-image: url('image.jpg');
background-size: cover;
opacity: 0.9;
    
    
    

    
    CSS Selectors
    
    
    
      
    /* Element Selector */
p { color: blue; }

/* Class Selector */
.highlight { background: yellow; }

/* ID Selector */
#header { height: 100px; }

/* Descendant Selector */
article p { margin-bottom: 1em; }

/* Child Selector */
ul > li { list-style: none; }
